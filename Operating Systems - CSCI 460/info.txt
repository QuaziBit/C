How to compile .c file and how to compile it with pthread
1: Compile basic .c file:      "gcc -c file.c" --> it will create <file.o>
2: build basic .exe file:      "gcc -o output_file_name file.o"
3: compile .exe with pthread:  "gcc -o output_file_name file.o -lpthread"
4: run .exe file:              "./output_file_name"


Extra:  gcc -S file.c
        vi file.s --> to see build file
